== Description ==

Plugin will find mentions of Bitcoin in your texts and automatically add a tooltip to it with actual price in USD and EUR. No need to do any setting.

You can also find a lot of other financial information and tools on <a href="https://freedomfinance.es/" rel="friend" title="freedomfinance.es">our website</a>.

This plugin was created with support of <a href="https://roxel.by/" rel="friend" title="Roxel Media">roxel.by</a>.
